Seletores CSS

Aplicando CSS a Diferentes Partes do HTML

1. O seletor de elemento mira nos elementos com base no nome da tag HTML.

  1. Seletores de classe miram nos elementos com base no valor do atributo class.
  2. Seletores de ID miram nos elementos com base no valor do atributo id.
  3. Seletores de atributo miram nos elementos com base em seus atributos e valores.
  4. O seletor universal mira em todos os elementos.
